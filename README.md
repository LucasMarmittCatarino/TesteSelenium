# Como usar

## Dependencias:
- ChromeDriver
```bash
# using chocolatey
choco install chromedriver
```

- Node
```bash
# using chocolatey
choco install nodejs
```

- run:
```bash
npm install
```

## Rodando o programa:
- Descobrir CPF:<br/>
```bash
node driver.js
```
- Macro resetar senha:
```bash
node macro.js
```
