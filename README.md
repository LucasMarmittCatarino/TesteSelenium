# Como usar

## Dependencias:
- ChromeDriver
```bash
# using chocolatey
choco install chromedriver
```

- Node
```bash
# using chocolatey
choco install nodejs
```

run:<br/>
- npm install<br/>

## Rodando o programa:
- Descobrir CPF:<br/>
```bash
node driver.js<br/>
```
Macro resetar senha:
```bash
node macro.js
```
